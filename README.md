# BrainBlitz

A study tool that turns content (uploaded PDFs/images or AI-defined topics) into quizzes, games, and learning materials.

## Project Documentation

- **[Project Overview](./BrainBlitz_Project_Overview.md)** - Complete specification and requirements
- **[Initialization Guide](./initializing.md)** - Technical setup and architecture decisions

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **UI:** React + Tailwind CSS
- **Backend:** Supabase (Postgres)
- **External APIs:** Mathpix (MMD extraction), LLM (content generation)
